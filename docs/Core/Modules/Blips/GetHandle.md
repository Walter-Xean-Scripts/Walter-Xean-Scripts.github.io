The primary functions for creating easy blips.

### Function
```lua
local Blips = WXSCore.Blips
Blips:GetHandle(id)
```

### Parameters
| Name | Type | Description |
| --- | --- | --- |
| id | integer | The blip ID returned from the functions. |

### Returns
| Name | Type | Description |
| --- | --- | --- |
| blip | integer | The blip handle. |
